# Sales Performance Analysis

This project analyzes Superstore Sales data to evaluate sales performance across regions, product categories, and time periods. The analysis includes data cleaning, KPI calculations, exploratory data analysis (EDA), and visualizations to identify sales trends, top-performing products, regional performance, and seasonal patterns.

## Objectives

* Clean and prepare sales data for analysis.
* Calculate key performance indicators (KPIs) such as Total Revenue, Total Orders, and Average Order Value.
* Analyze sales performance across regions and categories.
* Identify top-performing and low-performing products.
* Examine monthly sales trends and seasonality.
* Provide actionable business recommendations based on insights.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Git & GitHub

## Key Insights

* Regional sales comparison and performance analysis.
* Category-wise sales distribution.
* Top 10 and lowest-performing products.
* Monthly sales trend analysis for seasonality detection.
* Business recommendations to improve sales performance.

## Deliverables

* Jupyter Notebook with complete analysis.
* Visualizations and charts.
* Executive Summary Report.
* Business recommendations based on data-driven insights.
